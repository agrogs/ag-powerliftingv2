# Powerlifting Program Generator

A simple web app that generates structured powerlifting programs based on user inputs. The goal is to provide lifters with an easy way to create a balanced training plan focused on the **squat, bench press, and deadlift**.

## Features

* Generate structured weekly training programs
* Focus on the three main powerlifting movements
* Adjustable training inputs
* Simple browser-based interface
* No installation required

## How It Works

1. Open the app in your browser.
2. Enter your training information (such as training max, frequency, or preferences depending on the version).
3. The app generates a structured training program with sets, reps, and progression.

## Project Structure

```
/project-folder
  ├── index.html

```

The app runs entirely in the browser using HTML, CSS, and JavaScript.

## Running the App

Because the project is static, you can run it in several ways:

### Option 1: Open Locally

Simply open `index.html` in any modern web browser.

### Option 2: Deploy Online

You can deploy the project easily using static hosting platforms such as:

* Cloudflare Pages
* Netlify
* GitHub Pages

Upload the project folder and the app will run directly from the browser.

This project was created as a lightweight tool for lifters who want quick program ideas without needing complex software or spreadsheets.
